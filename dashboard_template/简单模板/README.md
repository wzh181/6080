# 简单响应式模板

最简化的响应式模板，只有2个文件。

## 文件

- `App.jsx` - 所有组件都在一个文件里
- `App.css` - 所有样式都在一个文件里

## 使用方法

1. 复制 `App.jsx` 和 `App.css` 到你的项目
2. 安装依赖：`npm install react react-dom react-router-dom`
3. 运行：`npm run dev`

## 响应式

- 桌面端：显示完整文字（Home | Game1 | Game2 | Game3）
- 移动端（≤800px）：显示缩写（H | G1 | G2 | G3）

就这么简单！
